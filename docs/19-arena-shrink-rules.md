# 19. Arena Shrink Rules (MVP)

작성일: 2026-03-04

## 1) 확정 방향
- 아레나는 시간 경과에 따라 **점진적으로 축소**한다.
- 아레나는 **최소 크기(min size)** 이하로는 줄어들지 않는다.

## 2) 확정 수치 (2026-03-04)
- 축소 시작 시점: **라운드 시작 30초 후**
- 축소 속도: **초당 반경 1.5% 감소**
- 최소 반경: **초기 반경의 35%**

## 3) 목적
- 후반 교전을 강제해 템포를 유지
- 넉백/위치 싸움의 가치를 높임
- 전장 소멸로 인한 억지 판정 방지(최소 크기 보장)

## 4) 구현 체크포인트
- 축소 시각화(경계선/바닥 이펙트)로 플레이어 인지성 확보
- 최소 반경 도달 후 추가 축소 중지
- Host 기준으로 축소 타이머를 권위 있게 관리하고 클라이언트에 동기화
