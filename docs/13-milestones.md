# 13. Milestones

## M0 - 문서 고정 (현재)
- One Pager, 조작/스킬/네트워크 초안 확정

## M1 - Vertical Slice 설계 완료
- 캐릭터 1(고정), 맵 1, 12키 로드아웃(QWER/ASDF/ZXCV), 승패 규칙 확정
- 네트워크 MVP 스택 확정
- 라운드 루프(종료→보상→상점→다음 라운드) 설계 확정
- 룸 설정(라운드 수 3/5/7, 점수판정) 반영

## M2 - Windows 구현 1차(오프라인 전투)
- 로컬에서 이동/스킬/피해/사망 동작

## M3 - 온라인 MVP(2~8인 룸)
- 방 생성/입장/매치 완료
- 경기 결과 화면
- 골드 지급 및 스킬 상점 루프 동작
- 산출물: Windows 테스트 빌드(내부 플레이테스트용)
- 매치메이킹 범위: 방 생성 + 초대 코드(랜덤 매칭 제외)

## M4 - 밸런스/UX 다듬기 (개인전 기준)
- DoT/쿨다운/피드백 조정
- 플레이테스트 반복

## M5 - 팀전(2v2) 확장
- 팀 배정/아군 판정/UI/HUD 확장
- 팀 승리 조건 및 종료 플로우 검증

## 각 마일스톤 공통 DoD(Definition of Done)
- 테스트 체크리스트 통과
- 치명 버그 우선순위 정리
- 다음 마일스톤 진입 조건 문서화
