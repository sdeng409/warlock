# 03. Controls & Camera

## 1) 입력 기본
- 이동: **우클릭 클릭 이동 (확정)**
- 스킬 키 범위: **QWER ASDF ZXCV (최대 12 슬롯)**
- 보조: **좌클릭(스킬 대상/방향 지정, 확정)**, Space(긴급 회피 후보)

## 2) 이동/조준 입력 결정
- **확정:** 우클릭 클릭 이동
- **확정:** 좌클릭으로 스킬 대상/방향 지정
- 선택 이유: 이동 입력과 조준 입력을 분리해 실수 입력을 줄이고, 전투 가독성을 높임
- 확장 방향: 추후 옵션으로 WASD + 마우스 조준 모드 추가 검토

## 3) 카메라
- 기본: 고정 탑뷰, 플레이어 추적
- 금지: 과한 화면 흔들림/줌 인아웃 남발
- 목표: 스킬 예고와 위험영역을 항상 인지 가능

## 4) UX 최소 요구
- 스킬 쿨다운 숫자+게이지
- 경계 밖 경고(화면 가장자리 붉은 비네트 + 경고음)
- 피격 방향 피드백


## 5) 타겟팅(3번 질문) 결정
- **확정:** 소프트 락온
- 기본 동작: 좌클릭으로 지정한 방향/대상을 우선하되, 근접 후보 대상에 약한 보정 적용
- 목표: 초보자 조작 피로를 낮추고, 클릭 이동 환경에서도 스킬 명중 체감을 안정화


## 6) 스킬 키 매핑 규칙(확정)
- 고정 키 범위: `QWERASDFZXCV`
- 플레이어는 원하는 스킬을 원하는 키에 배치 가능
- 캐릭터별 고정 스킬 구조는 사용하지 않음
