# 04. Combat Rules

작성일: 2026-03-04

## 1) 전투 철학
- 핵심 승리 수단: **넉백 + 위치 우위 + 경계 압박**
- 캐릭터는 동일, 빌드 차이는 스킬 조합에서 발생

## 2) 기본 규칙
- 라운드 승리: 마지막 생존 플레이어
- 리스폰: 없음 (사망 시 관전)
- 타겟팅: 소프트 락온

## 3) 경계 규칙
- 경계 밖: 유예 없음
- DoT: 1초마다 최대체력 5%
- 복귀 시 DoT 즉시 중단
- 악용 방지: 누적 타이머 즉시 리셋 금지, 3초 연속 복귀 시 리셋

## 4) 아레나 축소
- 시작: 30초
- 속도: 초당 반경 1.5% 감소
- 최소 반경: 초기 35%

## 5) 동점 규칙 (매치)
- 1순위: 라운드 1위 횟수
- 2순위: 마지막 라운드 순위
