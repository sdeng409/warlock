# 02. Core Loop

작성일: 2026-03-04

## 1) 매치 루프 (MVP)
1. 룸 생성/입장 (Host + Relay, 초대 코드)
2. 라운드 시작 카운트다운
3. 전투 진행 (생존전)
4. 라운드 종료 (순위 확정)
5. 순위 점수 + 골드 지급
6. 상점 페이즈(구매/슬롯 재배치)
7. 다음 라운드 시작
8. 최종 라운드 종료 후 누적 점수 우승자 결정

## 2) 라운드 상태 머신
- `Waiting` -> `RoundStart` -> `Combat` -> `RoundEnd` -> `Shop` -> `RoundStart`
- 마지막 라운드 종료 시 `MatchEnd`

## 3) 룸 설정(Host)
- 라운드 수: 3 / 5 / 7 (기본 5)
- 플레이어 수: 2~8
- 모드: FFA (MVP)
