# 10. Art Style Guide

작성일: 2026-03-04

## 1) 방향
- 탑뷰 가독성 최우선
- 실루엣/색 대비로 스킬 구분
- 과도한 디테일보다 판정 명확성 우선

## 2) 전장/캐릭터
- 배경 명도는 중저채도
- 플레이어/적/위험영역은 높은 대비 색상
- 축소 경계선은 한눈에 보이게 강한 컬러 사용

## 3) VFX 원칙
- 넉백 시작/끝 타이밍이 보이도록 설계
- AoE 범위 예고는 최소 0.3~0.5초 확보

## 4) IP 가드레일
- Warcraft 원본 캐릭터/아이콘/사운드/명칭 직접 사용 금지
- "감성"만 계승, 에셋은 신규 제작/합법 리소스 사용
