# 24. Acceptance Test Spec (MVP)

작성일: 2026-03-04

## A. 매치 루프
- [ ] 5라운드 기본 설정에서 `RoundStart->Combat->RoundEnd->Shop`가 끊김 없이 반복
- [ ] 최종 우승이 누적 점수 기준으로 정확히 계산
- [ ] 동점 시 1위 횟수 -> 마지막 라운드 순위 규칙 적용

## B. 스킬/로드아웃
- [ ] 12키 슬롯 자유 배치 동작
- [ ] 미소유 스킬 장착 불가
- [ ] 쿨다운 중 재시전 거부

## C. 경제/상점
- [ ] 라운드 종료 후 순위 골드 지급 정확
- [ ] 상점 구매 시 골드 차감/소유 반영 정확
- [ ] 리롤/재판매 버튼/경로 없음

## D. 경계/축소
- [ ] 장외 DoT 1초 5% 적용
- [ ] 복귀 즉시 DoT 중단 + 누적 타이머 3초 규칙 적용
- [ ] 30초 후 축소 시작, 1.5%/s, 35% 최소 반경 유지

## E. 네트워크
- [ ] 2~8인에서 이벤트 동기화 일관성 유지
- [ ] Host 재시작/연결 끊김 시 안전 종료 또는 복구 동작 명확
