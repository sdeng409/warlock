# 06. Arena Boundary & DoT Rules

작성일: 2026-03-04

## 1) 확정 규칙 (MVP)
- 경계 밖으로 나가면 **유예 없이 즉시 위험 상태**
- 피해는 **1초마다 최대체력 5%**
- 경계 안으로 복귀하면 **DoT 즉시 중단**
- 사망 시 **리스폰 없이 관전 전환**

## 2) 설계 의도
- 전장 이탈 플레이를 빠르게 억제
- 매치 템포를 유지하고 교전 밀도를 높임
- 원작 감성 정렬: 장외 즉사 대신 누적 위험 누적으로 탈락 유도

## 3) UX 최소 요구
- 화면 가장자리 경고(붉은 비네트)
- 경고 사운드(DoT 상태에서 반복)
- 체력 감소 피드백(틱 타이밍이 체감되도록)

## 4) 악용 방지 규칙(확정)
- 경계 밖 누적 타이머를 운영한다.
- 경계 안으로 복귀해도 누적 타이머는 즉시 0으로 초기화하지 않는다.
- 복귀 후 **연속 3초** 동안 경계 안에 머물렀을 때만 누적 타이머를 초기화한다.
- 3초 이내 재진입 시, 이전 누적 상태를 이어서 DoT를 적용한다.
