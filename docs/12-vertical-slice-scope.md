# 12. Vertical Slice Scope (MVP)

작성일: 2026-03-04

## In Scope
- 2~8인 FFA
- Host + Relay 방 생성/초대 코드
- 클래스리스 12키 스킬 매핑
- 초기 스킬 풀 12개
- 라운드 순위 점수 + 골드 보상 + 상점 구매
- 아레나 축소 + 경계 DoT + 누적 타이머

- Dedicated 서버
- 팀전(2v2) (v0.2)
- 랭크/MMR/시즌
- 리롤/재판매
