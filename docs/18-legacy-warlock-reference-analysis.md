# 18. Legacy Warlock Reference Analysis (User Source 기반)

작성일: 2026-03-04

## 1) 기준 레퍼런스
아래 분석은 사용자가 직접 제공한 Warcraft III Warlock 설명을 1차 기준으로 작성.

## 2) 원작 감성 핵심 (반드시 유지할 요소)
1. **라운드 생존전**: 마지막 1인/팀 생존이 라운드 승리
2. **장외 처치 중심 전투**: 직접 체력 감소보다 **넉백으로 플랫폼 밖으로 밀어내는 플레이**
3. **아레나 축소 압박**: 시간이 갈수록 발판/전장이 좁아져 긴장도 상승
4. **라운드 후 성장 루프**: 순위/처치 보상 골드로 스킬 구매/강화
5. **빌드 다양성**: 공격/방어/유틸 스킬 조합으로 플레이스타일 분화

## 3) 현재 warlock 기획과 정합성
- 이미 정합: 클래스리스 스킬 조합, 라운드+상점, 빠른 교전 템포
- 보강 필요: "넉백 기반 장외 처치"를 더 중심 규칙으로 강조
- 보강 필요: "아레나 축소" 규칙을 정량화(축소 시작 시점/속도/최소 반경)

## 4) 적용 방안 (MVP 친화)
### Keep
- 멀티 라운드 + 순위 골드 + 상점
- 클래스리스 12키 자유 매핑

### Add
- 넉백 계열 스킬 비중을 스킬풀의 핵심 축으로 배치
- 라운드 타이머 기반 아레나 축소 규칙 도입

### Decide Next (확정 반영)
- 장외 처리 규칙 확정: **A안(DoT 누적 방식 유지)**
- 사용자 회고 반영: 원작 체감은 즉사보다 누적 피해형 탈락에 가까움
